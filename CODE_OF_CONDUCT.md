# Code of Conduct

## Our Pledge

We want this project to be useful, respectful, and welcoming to people building
agent-assisted workflows across different tools and teams.

## Expected Behavior

- Be kind, specific, and constructive.
- Assume good intent while still naming risks clearly.
- Respect privacy, licenses, and provenance.
- Keep discussions focused on reusable workflow and engineering quality.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private data, secrets, or identifying information without consent.
- Submitting project-specific or proprietary content as generic material.
- Ignoring license or attribution requirements.

## Enforcement

Maintainers may edit, close, or remove contributions that violate this code of
conduct, especially content that risks privacy, safety, or license compliance.
